# re:AniDesk (Custom Build)

[![VirusTotal](https://img.shields.io/badge/VirusTotal-1/70_Safe_(False_Positive)-success?logo=virustotal)](https://www.virustotal.com/gui/file/5786cce8c4ff1a617cc2acf0f1e8f710fdb3415403aa5bb0ae476448125dad4c/detection)

![Интерфейс AniDesk](public/assets/images/screenshot.png)

Неофициальный десктопный клиент для просмотра аниме на базе AniDesk (Anixart API).
Эта версия включает эксклюзивные визуальные изменения, оффлайн-режим и расширенные возможности плеера, а также встроенный обход блокировок (CDN-прокси) для бесперебойной загрузки контента.

## Важно:
- При установке проекта и появлению .exe в системе будет исключительно оригинальное название проекта.
- Все логи об ошибках и прочее пока что летят только официальному разработчику AniDesk. В будущем будет исправлено или изменено.
- Внутреннее безшовное обновление (update.exe) и авто процесс обновления пока что тоже работает на стороне оригинального разработчика.
---

## 🔥 Ключевые нововведения (Отличия от оригинала)

### 📺 Умный плеер
- **Расширенное управление:** Поддержка скорости `1.25x` и режима «Оригинальное» соотношение сторон.
- **Память плеера:** Автоматическое запоминание выбранной озвучки, источника видео и позиции просмотра для каждой серии.
- **Бесшовный просмотр:** Мгновенный переход между сериями без искусственных задержек.

### 📥 Оффлайн-загрузчик
- **Скачивание серий:** Полноценный менеджер загрузок с поддержкой потоков HLS/m3u8 и прямых MP4.
- **Локальная библиотека:** Отдельный раздел "Оффлайн" для управления скачанными тайтлами.
- **Собственный протокол:** Локальный плеер работает напрямую с диском через безопасный протокол `anidesk-offline://`.

### 🎨 Интерфейс и UI/UX
- **Система оценок:** Интерактивное меню для быстрого выставления оценок тайтлам (от 1 до 5 звезд) в один клик.
- **Умная навигация:** Точное запоминание и восстановление позиции скролла при возврате к длинным спискам (избранное, планы и др.).
- **Визуальный апгрейд:** Чёткие векторные иконки, классические hover-эффекты и press-анимации карточек.
- **Отклик и плавность:** Отключено размытие карточек (GPU-ускорение) и внедрена параллельная загрузка постеров без фризов.

## 📥 Установка

1. Перейдите в раздел **[Releases](https://github.com/Ar3sSs-dev/re-AniDesk/releases)**.
2. Скачайте последнюю версию (установщик `anidesk-*-win32.exe` из раздела `Assets`).
3. Запустите установщик.

### 💻 Быстрая установка (PowerShell)

Для скачивания и запуска установщика версии `2.1.1` одной командой, выполните в PowerShell:

```powershell
Invoke-WebRequest -Uri "https://github.com/Ar3sSs-dev/re-AniDesk/releases/download/v2.1.1/anidesk-2.1.1-win32.exe" -OutFile "$env:TEMP\anidesk_setup.exe"; Start-Process "$env:TEMP\anidesk_setup.exe"
```
