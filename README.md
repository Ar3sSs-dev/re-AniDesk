# AniDesk (Custom Build)

Неофициальный и улучшенный десктопный клиент для просмотра аниме, основанный на Anixart API. 

Эта версия содержит эксклюзивные визуальные улучшения, современные контурные иконки, исправленную геометрию интерфейса, а также **встроенный обход блокировок провайдера** (через быстрый CDN-прокси), благодаря которому обложки и картинки загружаются мгновенно и без ошибок.

---

## ✨ Что изменено / добавлено?

- **Оффлайн-загрузчик:** Полноценный менеджер скачивания серий (HLS/m3u8 и MP4), своя оффлайн-библиотека и воспроизведение без интернета через локальный протокол `anidesk-offline://`.
- **Оптимизация кэша и файловой системы:** Искоренены конфликты Windows при записи (EBUSY), добавлен асинхронный запуск (никаких зависаний при старте из-за очистки картинок), параллельные загрузки постеров работают идеально (Stream Lock fix).
- **Тотальная стабильность плеера:** Устранены утечки памяти при переходе между сериями (корректное уничтожение `hls.js`), исправлено залипание ползунка перемотки, убраны скрытые 1.5-секундные задержки перед запуском видео с Libria/Sibnet.
- **Обход блокировок картинок:** Изображения с `anixmirai` теперь загружаются через глобальный кэширующий CDN Google/weserv. Картинки больше не "отваливаются" по таймауту у пользователей из СНГ.
- **Плавный UI без размытий:** Аппаратное ускорение (GPU) для карточек аниме — больше никакого "мыла" при наведении. Восстановлены красивые hover-эффекты и press-анимации.
- **Новый визуал и анимации:** Заменены старые мыльные иконки на строгие векторные (Lucide Icons 24x24). Добавлены плавные CSS-анимации выезжающих панелей и кнопок бокового меню. Выровнены отступы в сетке тайтлов.
- **Отказоустойчивость сети:** Если API Anixart падает или не отвечает, приложение больше не крашится белым экраном, а корректно отлавливает ошибки и показывает их пользователю.

## 📥 Как скачать и установить?

Скачать готовую к использованию программу можно прямо из этого репозитория:

1. Перейдите в раздел **[Releases (Релизы)](https://github.com/Ar3sSs-dev/AniDesk-App/releases)** справа.
2. Найдите последнюю опубликованную версию.
3. Скачайте архив с программой (или файл-установщик) из раздела `Assets`.
4. Распакуйте в удобное место и запустите `AniDesk.exe`!

> **Примечание:** Если вы уже пользуетесь старой версией AniDesk, вы можете просто скачать обновленный файл `app.asar` и заменить им старый по пути `resources/app.asar` в папке с вашей программой.

## 💻 Быстрая установка (Для продвинутых пользователей)

Если вы не хотите скачивать установщик вручную, вы можете установить программу одной командой через **Windows PowerShell**. 

Эта команда автоматически скачает последнюю версию установщика во временную папку и запустит его. Для этого откройте PowerShell (Пуск -> PowerShell) и скопируйте туда следующий код:

```powershell
Invoke-WebRequest -Uri "https://github.com/Ar3sSs-dev/AniDesk-App/releases/download/v1.0.0/anidesk-0.0.1-beta.7-win32.exe" -OutFile "$env:TEMP\anidesk_setup.exe"; Start-Process "$env:TEMP\anidesk_setup.exe"
```
