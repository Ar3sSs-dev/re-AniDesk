# re:AniDesk (Custom Build)

[![VirusTotal](https://img.shields.io/badge/VirusTotal-1/70_Safe_(False_Positive)-success?logo=virustotal)](https://www.virustotal.com/gui/file/d32574d85a82f2a39fc9a88ca194cd05381f3fabc5d8f335e7669a72ea2c499c/detection)

![Интерфейс AniDesk](public/assets/images/screenshot.png)

Неофициальный десктопный клиент для просмотра аниме на базе AniDesk (Anixart API).
Эта версия содержит встроенный обход блокировок провайдера (CDN-прокси) для загрузки медиа-контента, исправления интерфейса и расширенный функционал плеера.

---

## Основные возможности и отличия

### 📺 Плеер
- Мгновенный переход между сериями.
- Поддержка скорости `1.25x` и режима «Оригинальное» соотношение сторон.
- Быстрый отклик горячих клавиш (в т.ч. `Esc`) и управления звуком.
- Автоматическое запоминание позиции просмотра в серии.

### 📥 Оффлайн-режим
- Встроенный загрузчик для скачивания серий.
- Раздел "Оффлайн" для управления скачанными тайтлами.
- Воспроизведение локальных файлов напрямую через встроенный плеер.

### ⚡ Производительность
- Устранены ошибки EBUSY при записи кэша и утечки памяти при работе `hls.js`.
- Стабильная работа при недоступности серверов источника (перехват ошибок).
- Плавная параллельная загрузка обложек без зависаний.

### 🎨 Интерфейс (UI/UX)
- Восстановлены CSS-эффекты (hover, press), отключено размытие карточек (GPU-ускорение).
- Интерактивное выставление оценок тайтлам (1-5 звезд).
- Чёткие векторные иконки.

## Установка

1. Перейдите в раздел **[Releases](https://github.com/Ar3sSs-dev/re-AniDesk/releases)**.
2. Скачайте последнюю версию (архив или установщик из раздела `Assets`).
3. Распакуйте/установите и запустите `re-AniDesk.exe`.

### Быстрая установка (PowerShell)

Для скачивания и запуска установщика версии `2.1.1` выполните в PowerShell:

```powershell
Invoke-WebRequest -Uri "https://github.com/Ar3sSs-dev/re-AniDesk/releases/download/v2.1.1/anidesk-2.1.1-win32.exe" -OutFile "$env:TEMP\anidesk_setup.exe"; Start-Process "$env:TEMP\anidesk_setup.exe"
```
