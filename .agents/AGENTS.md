# Re:AniDesk — Локальный контекст

## Стек
- **Frontend:** Svelte 4 / Svelte 5 (смешанный) + Rollup
- **Backend:** Electron, Node.js, JS/TS
- **Сборщик:** Electron Forge, Rollup

## Архитектура
- `src/main.js` — точка входа главного процесса Electron
- `src/app/` — исходный код рендерера (Svelte компоненты)
- `forge.config.js` — конфигурация упаковщика Electron Forge
- `package.json` — метаданные проекта, скрипты сборки и зависимости

## Баги
- [x] Ошибки бесконечного лоадинга из-за отсутствия отмены устаревших fetch-запросов (решено через AbortController)
- [x] Проблема с голосованием/оценками в UserRating.svelte (решено добавлением паузы и получением токена напрямую)
